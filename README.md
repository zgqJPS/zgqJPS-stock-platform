# 股票数据分析平台

基于 Flask 和 AKShare 的股票数据分析平台，提供涨停数据分析、砸盘系数计算、连板矩阵可视化等功能。

## 功能特性

- 📈 涨停数据采集和分析
- 📊 砸盘系数计算
- 🎯 交易机会分析
- 📋 连板矩阵可视化
- 📤 数据导出功能

## 技术栈

- **后端**: Flask, Pandas, AKShare
- **前端**: HTML, CSS, JavaScript
- **图表**: Matplotlib
- **部署**: Render / Heroku
